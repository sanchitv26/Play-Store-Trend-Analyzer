# Play Store Trend Analyzer

An AI-powered tool to analyze Google Play Store reviews and generate trend analysis reports.

## Features

- **Daily Batch Processing**: Processes reviews day by day
- **Topic Extraction**: Identifies common issues, feedback, and requests
- **Trend Analysis**: 30-day rolling trend reports
- **Multiple Output Formats**: CSV and Excel reports
- **Visualization**: (Optional) Charts and graphs
